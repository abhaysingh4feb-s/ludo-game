## Game Rules

- Each player has **4 tokens** starting in their base
- Roll **6** to move a token out of base
- Tokens move clockwise around the 52-cell track
- Land on an opponent's single token to **send it back to base**
- Two of your tokens on the same cell form a **block** (immune to capture, opponents can't pass)
- Rolling **three consecutive 6s** skips your turn
- First player to get all 4 tokens **home** wins

### Bonus Turns

| Action | Reward |
|--------|--------|
| Roll a 6 | Extra turn |
| Kill an opponent token | Extra turn |
| Move a token to home | Extra turn |

> Bonus turns do **not** stack — only one extra turn per move.
